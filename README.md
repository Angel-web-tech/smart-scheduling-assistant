#  Smart Scheduling Assistant

An intelligent meeting scheduling system that automatically detects conflicts, finds optimal free slots, and improves productivity using smart suggestions.

Built with Python and Streamlit.

---

## Features

Add / Delete Meetings (CRUD)  
Conflict Detection  
Smart Slot Suggestions (AI scoring logic)  
Calendar View  
Task Manager  
SQLite Database Storage  
CSV Export  
Time Usage Analytics  
Buffer Time & Lunch Avoidance  

---

## Smart Logic

The system:
- Analyzes busy slots
- Applies buffer time between meetings
- Avoids lunch hours
- Scores time slots (morning preferred, late avoided)
- Suggests best meeting time automatically
